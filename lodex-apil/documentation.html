Documentation

Lodex (« Linked Open Data EXperiment ») est un outil permettant :

  • de publier des jeux de données (.csv, .tsv, .xml, .json, etc.) dans des formats du web sémantique (JSON-LD, N-Quads), et de les manipuler dans une interface d'administration dédiée ;
  • d'exposer ces données dans un rapport web dynamique se présentant sous la forme d’un tableau de bord comprenant des graphiques.
Chaque graphique est visible sur la page d’accueil mais également sur une page personnalisée accessible en cliquant sur le bandeau « VOIR LES DÉTAILS » (en bas de chaque graphique).
A ces graphiques sont associées des facettes permettant de croiser les informations. Ces facettes se trouvent à droite sur la page personnalisée.
Pour croiser les informations d’un graphique avec les informations d’une ou plusieurs facettes, il faut sélectionner les éléments qui peuvent être inclus (en cochant la ou les case(s) située(s) à gauche des éléments) ou exclus (en cochant également la case « exclure » au-dessus des éléments). Les graphiques se mettent à jour. Les critères de croisement apparaissent au-dessus du graphique. Les publications répondant aux critères du croisement sont proposées sous forme d’un tableau en bas de la page.
Dans ce rapport, il est également possible de visualiser les notices des publications :
  • sous forme d’un tableau :
    • dans « Documents » en bas de la page d'accueil
    • ou sur les pages personnalisée des graphiques (en bas)
  • sous forme détaillée en cliquant sur :
    • les titres des publications dans « Documents » (en bas de la page d'accueil)
    • les numéros des notices dans les tableaux des pages personnalisée des graphiques (en bas).
La documentation complète de cet outil est accessible à l'adresse suivante : https://lodex.gitbooks.io/lodex-user-documentation/
